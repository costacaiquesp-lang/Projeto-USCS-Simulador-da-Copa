# Projeto-USCS-Simulador-da-Copa
Simulação de Copa do Mundo feita com HTML, CSS e JavaScript. O usuário faz cadastro e participa de jogos em fases de grupos, quartas, semifinal e final. O sistema valida respostas, evita erros e define os classificados até chegar ao campeão exibido na tela.
